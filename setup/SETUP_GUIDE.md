# Hướng dẫn cài đặt Website Quản Lý Phòng Trọ

## Yêu cầu
- XAMPP
- PHP 8.x
- MySQL

## Các bước cài đặt
1. Copy thư mục src vào htdocs của XAMPP
2. Bật Apache, MySQL trong XAMPP 
3. Import file quanly_phongtro.sql vào phpMyAdmin
4. Kiểm tra cấu hình file db_connect.php trong thư mục src đúng tên file.sql chưa
5. Truy cập http://localhost:8080/src
